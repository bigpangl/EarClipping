尝试完成EarClipping 算法


关键点概念

- EarCliping 算法中，关于凸角的判断，需要有一个前置条件，就是二维坐标点

- 凸角判断需要根据顺/逆时针分开判断，green 公式可以用于判断二维坐标是顺时针还是逆时针

- 处理三维空间中的多边形（处于一个平面），尝试将降维处理，分割后，再升维。
    
    所以此处需要定义平面,有x轴和y轴单位长度法向量。如果没有已知的单位长度法向量,需要从其他途径提供这两者


整体逻辑：

1. 降维处理

1. 顺/拟时针处理

1. 点顺序颠倒 / or not ，强制为拟时针

1. EarClipping 算法:
    
    1. 寻找凸角 且其他顶点不在此三个点构成的三角形中（角度法判断）
